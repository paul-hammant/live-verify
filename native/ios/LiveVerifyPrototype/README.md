# LiveVerifyPrototype (Xcode project)

This folder contains a minimal iOS app + unit tests.

- App: `native/ios/LiveVerifyPrototype/LiveVerifyPrototype.xcodeproj`
- Sources: `native/ios/LiveVerifyPrototype/LiveVerifyPrototype/`
- Tests: `native/ios/LiveVerifyPrototype/LiveVerifyPrototypeTests/`

Open the project in Xcode and run on a real device (camera required). Unit tests are focused on the pure logic (verify-line parsing, normalization, SHA-256, and response interpretation).
